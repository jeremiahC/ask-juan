How It Works

One prompt.
Your whole team moves.

Type /juan:route followed by what you need — in plain English, Tagalog, or Taglish. Juan reads your project context and dispatches the right specialist.

Claude Code
$ /juan:route I want to build a user authentication feature
Juan

Got it. Reading project context... (.juan/context.md)

Routing to: Rica → write specs · Carlo → plan architecture · Nico + Rex → build in parallel

Spawning agents...

$ /juan:route Give me updates on today's sprint
Juan

Pulling sprint status...

Routing to: Tina → QA summary · Dino → test report · Gab → code review status

Compiling sprint update...

$ /juan:route Give me data about our sales this month
